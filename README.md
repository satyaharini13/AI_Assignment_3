# UGV Path Planning Assignment

## Description
This project implements:
- Dijkstra Algorithm for city road networks
- A* algorithm for grid navigation
- Dynamic obstacle handling

## Features
- 70x70 grid navigation
- Three obstacle densities
- Dynamic replanning

## How to Run
python experiments/dijkstra_test.py
python experiments/static_test.py
python experiments/dynamic_test.py

## Output
- Shortest paths
- Path length
- Execution time
